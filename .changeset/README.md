## React Day Picker notes

This repo keeps the publishable packages versioned together:

- `react-day-picker`
- `@daypicker/*`

Private apps are ignored by Changesets. `shadcn-app-v9` intentionally depends on
the published v9 package, so Changesets may print a dependency graph warning
about that package during `status` or `version`. Keep
`bumpVersionsWithWorkspaceProtocolOnly` disabled so the calendar package peer
dependencies on `react-day-picker` are updated during versioning.

The current package version is a `next` prerelease. To continue the `next` train
with Changesets, enter prerelease mode before running `changeset version`:

```sh
pnpm changeset pre enter next
```

Publishing only happens when `pnpm changeset publish` is run.
